# Week 2 Homework

### 1. Function template:  Maximum of Two Values
Write a function template named `myMax` that returns the larger of two values.
#### Requirements:
- Edit file myMax.cpp.
- The function should work for at least int, double, and string
- Use a single template definition


### 2. Class templates: Template-Based Box Class ###
Create a class template named Box that stores a single value of any type.
#### Requirements:
- Edit Box.cpp for this task.
- Implement member functions:
    - Constructor
    - setValue(T v)
    - getValue()
    - print()

### 3. Template and Arrays: Template Function for Array Sum
Write a function template named arraySum that computes the sum of elements in an array.
#### Requirements:
- Complete TO DO tasks in ArraySum.cpp - template that computes sum of elements in the array.
- Assume + operator is supported for the type
- Example:
  - int arr1[] = {1, 2, 3, 4};
  - double arr2[] = {1.1, 2.2, 3.3};

cout << arraySum(arr1, 4) << endl;
cout << arraySum(arr2, 3) << endl;

Implement ClassPair class methods in  ClassPairTemplate.tpp
